【現場調査チェックリスト PWA版 v3・ロゴ対応】

■今回の追加
・アプリ内に「会社ロゴ設定」を追加
・PNG、JPG、WebP画像を端末内に登録可能
・簡潔報告書の左上に登録ロゴを自動表示
・ロゴ未登録時は「株式会社ロジック」の文字ロゴを表示
・登録ロゴはIndexedDBへ保存され、次回以降も使用可能

■ロゴ登録方法
1. アプリを開く
2. 「会社ロゴ設定」まで移動
3. ロゴ画像を選択
4. 「ロゴを端末に登録」を押す
5. 「簡潔報告書（約2枚）」を表示

■GitHub Pagesの更新
このZIP内の5ファイルを、現在のリポジトリへ上書きアップロードしてください。

index.html
manifest.webmanifest
sw.js
icon-192.png
icon-512.png

アップロード後に古い画面が出る場合は、
ブラウザを強制再読み込みし、ホーム画面アプリを完全終了して開き直してください。
