【現場調査チェックリスト PWA版】

■PWA版の機能
・スマホのホーム画面へアプリとして追加
・インストール後はオフライン起動可能
・入力内容を端末内に保存
・各箇所の写真も端末内に保存
・CSV出力
・印刷画面からPDF保存
・駐車、搬入、作業時間、寸法、安全、設備、写真項目を収録

■重要
PWAは、ZIPをスマホで直接開くだけではインストールできません。
HTTPS対応のウェブサーバーへ、このフォルダ内のファイルをすべて公開する必要があります。

公開するファイル：
・index.html
・manifest.webmanifest
・sw.js
・icon-192.png
・icon-512.png

■公開方法の例
1. GitHub Pages
2. Netlify
3. Cloudflare Pages
4. 自社のHTTPS対応ウェブサーバー

■スマホへの追加
Android：
Chromeで公開URLを開き、「ホーム画面に追加」ボタンを押します。
表示されない場合は、Chromeメニューの「アプリをインストール」を選択します。

iPhone：
Safariで公開URLを開き、共有ボタンから「ホーム画面に追加」を選択します。

■写真保存
「端末に保存」を押すと、入力文字と選択した写真をブラウザのIndexedDBへ保存します。
写真が多すぎる場合、端末容量の制限で保存に失敗することがあります。

■PDF保存
「印刷・PDF保存」を押し、端末の印刷画面からPDFとして保存します。
写真付きのPDFは写真枚数や端末性能によって処理に時間がかかる場合があります。
